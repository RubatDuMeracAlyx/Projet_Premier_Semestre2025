notre jeu est un bomber-man
le but est donc pour chaque joueur de casser les caisses afin d'ateindre l'autre joueur et le faire exploser.

les controles du joueur 1 (rouge en haut à gauche) sont les fleches directioneles et shift pour poser une bombe

les controles du joueur 2 (vert en bas à droite) sont WASD et espace pour poser une bombe

une fois le projet instaler il vous faudra faire un clicdroit sur le dossier src et ensuite "ajouter comme source"
ensuite redéclarer fungraphics comme library
et enfin importez un JDK.

ensuite pour lancer le jeu il vous faudra lancer MazeGame qui se trouve dans le dossier src
